"Yet another presentation" - это JavaScript-плагин для демонстрации презентаций с поддержкой jQuery. Этот плагин может использоваться в vanilla-форме yaPresentation(DOMel, options), либо $(el).yaPresentation(options).

Для работы плагина не нужны зависимости; jQuery-запуск является лишь оберткой для функций плагина.

Для продолжения кликните еще раз...

Особенности кода

Репозиторий проекта размещен на Github, при разработке были использованы следующие технологии:

  • Grunt
  • Jasmine
  • Travis CI

Проект совместим с IE9+, JavaScript-код покрыт тестами более чем на 90%. В сжатом виде JS-файл занимает 6кб.
